This project implements a Nuxeo iPhone client. It's just a proof of concept.

Actually, it's not specific to Nuxeo, as it currently only uses the CMIS
protocol.

You need to install CMIS 1.0 support on the Nuxeo server side to test the
client.

Information on how to do this are provided here:
http://doc.nuxeo.org/xwiki/bin/view/Main/CMIS
